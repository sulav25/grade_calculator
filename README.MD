# Grade Calculator

## Description
This Python application allows users to input grades, calculate the final grade based on weighted categories, and display the results with color-coded letter grades. The application uses a simple GUI built with **tkinter**.

## Features
- Input grades manually for different categories.
- Assign weights to each category to calculate a weighted final grade.
- Display the final grade with a color code (green for high, yellow for average, and red for low grades).
- User-friendly interface with **tkinter**.

## Requirements
- Python 3.8 or higher
- tkinter (included in Python standard library)
